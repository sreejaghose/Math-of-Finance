# Math-of-Finance
Mathematical finance models and algorithms implemented in Python, covering portfolio theory, derivatives pricing, fixed income, and stochastic processes.

This repository contains implementations of concepts and models from mathematical finance, including:

- Annuity Workbook
- Coupon Bond Pricing (Clean & Dirty Prices)
- Option Pricing
- Random Walks
- Two Asset Portfolio (shape widget with slider)
- Efficient Frontier 

The goal of this repository is to document quantitative finance concepts through code and numerical examples.
